## Description
This is a simple React.JS budget tracker application. The main purpose behind building this application was to get some familiarity with the use, purpose and deployment of Personal Web Applications. 

## Installation
Because this is a personal web application, to install this application, visit the deployed page, if youre using a compatible browser then the browser should prompt you to install this application on page load.

## Usage
There are up and down arrows that you can use to increase or decrease the total amount.
